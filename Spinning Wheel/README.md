# Spinning Wheel

The SpinningWheel library for Android lets you customize the spinning wheel component easily. Here's a simple breakdown of its attributes:

1. wheel_colors: Choose colors for the wheel segments.
2. wheel_stroke_color: Set the color of the segment borders.
3. wheel_stroke_width: Adjust the thickness of the segment borders.
4. wheel_items: Add text content to each segment.
5. wheel_text_size: Change the size of the text.
6. wheel_text_color: Pick a color for the text.
7. wheel_arrow_color: Define the color of the selection arrow.
8. wheel_arrow_width: Adjust the width of the arrow.
9. wheel_arrow_height: Set the height of the arrow.
